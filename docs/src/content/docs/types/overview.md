Sparkle provides comprehensive TypeScript definitions for all components, utilities, and design tokens.

## Core Types

### Component Props

- `ButtonProps`
- `InputProps`
- `ThemeProviderProps`

### Theme Types

- `ThemeConfig`
- `DesignToken`
- `ColorToken`

### Utility Types

- `HTMLProperties`
- `BaseProps`
- `VariantProps`

## Type Safety

All Sparkle APIs are fully typed for the best developer experience.
